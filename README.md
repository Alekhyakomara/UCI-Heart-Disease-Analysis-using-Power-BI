# ❤️ Heart Disease Analysis using Power BI

## 📌 Project Overview

This project analyzes the **UCI Heart Disease Dataset** using **Power BI** to identify key risk factors associated with heart disease. Interactive dashboards and visualizations provide insights into patient demographics, medical attributes, and disease patterns to support data-driven healthcare decisions.

## 🎯 Objectives

* Analyze heart disease risk factors using visual analytics.
* Clean and transform healthcare data for reporting.
* Build an interactive Power BI dashboard.
* Identify patterns based on age, gender, cholesterol, blood pressure, and chest pain.
* Support healthcare professionals with meaningful insights.

## 🔍 Key Insights

* Male patients showed a higher number of heart disease cases.
* Chest Pain Types 2 and 3 had a stronger relationship with heart disease.
* Higher cholesterol and blood pressure were associated with increased disease risk.
* Exercise-induced angina was a strong indicator of heart disease.
* Most heart disease cases occurred among patients aged 30–40 years.
* Heart rate and Oldpeak values helped identify exercise-related cardiac stress.
